## üéØ Usage Tips

### Getting Better Responses
1. **Be specific:** "Tell me about Indian food" vs "Tell me about food"
2. **Provide context:** "I'm feeling sad because..." vs "I'm sad"
3. **Ask follow-ups:** Keep the conversation flowing
4. **Use her name:** "Priya, what do you think about..."

### Memory Optimization
1. **Use !remember:** Save important info explicitly
2. **Regular conversations:** More context = better memory
3. **Clear old data:** Use !forget occasionally for fresh start
4. **Be consistent:** Use same names for people/places

### Voice Chat Tips
1. **Clear audio:** Use good microphone for better recognition
2. **Natural speech:** Talk normally, don't over-enunciate
3. **Wait for response:** Let her finish before speaking again
4. **Use wake word:** Say "Priya" to get attention

### Performance Tips
1. **Use local models:** Install Ollama for faster responses
2. **Multiple API keys:** Add backup providers for reliability
3. **Monitor health:** Check !health regularly
4. **Restart occasionally:** Fresh start can improve performance
