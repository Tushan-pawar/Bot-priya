# Voice Best Friend Discord Bot - Priya

Meet Priya, your intelligent 23-year-old Indian best friend who talks with you via voice using 100% free, unlimited local AI models.

## Setup

1. **Install Ollama** (for AI brain):
   - Download from https://ollama.ai
   - Run: `ollama pull llama3.2`

2. **Install FFmpeg** (for audio):
   - Download from https://ffmpeg.org
   - Add to PATH

3. **Install Python dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

4. **Create Discord Bot**:
   - Go to https://discord.com/developers/applications
   - Create bot, enable voice intents
   - Copy token to `.env` file

5. **Run**:
   ```bash
   python bot.py
   ```

## Usage

1. Join a voice channel
2. Type `!join` - bot joins
3. Just talk! Bot listens, responds with voice
4. Type `!leave` - bot leaves
